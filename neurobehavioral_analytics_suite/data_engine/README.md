<!-- ABOUT THE PACKAGE -->
## About The Package

The `data_engine` package within the NeuroBehavioral Analytics Suite (NBAS) defines and manages the core data processing and management functionalities for a project. The key component of this package is:

- **DataEngine**: Serves as the primary functional handler for a project, initializing the function engine, setting the function range, and executing the function engine.

### DataEngine
The `DataEngine` class is responsible for initializing the function engine, setting the function range, and executing the function engine. It handles data extraction, processing, and logging for the project.
