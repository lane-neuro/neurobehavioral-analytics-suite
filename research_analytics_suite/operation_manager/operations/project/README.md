<!-- ABOUT THE SUBPACKAGE -->
## About The Subpackage

The `project` subpackage within the `operation_manager/operations` module defines and manages operations related to project management in the Research Analytics Suite (RAS). The key components of this subpackage include:

- **CreateProjectOperation**: Handles the creation of new projects, initializing necessary data structures and settings.
- **LoadProjectOperation**: Manages the loading of existing projects, setting up the environment and data for continued work.
- **SaveProjectOperation**: Handles the saving of projects, ensuring that all data and settings are stored correctly for future use.

### CreateProjectOperation
The `CreateProjectOperation` class handles the creation of new projects within the RAS. It initializes necessary data structures and settings to set up a new project.

### LoadProjectOperation
The `LoadProjectOperation` class manages the loading of existing projects. It sets up the environment and data to continue working on a previously saved project.

### SaveProjectOperation
The `SaveProjectOperation` class handles the saving of projects within the RAS. It ensures that all data and settings are stored correctly, allowing the project to be resumed in the future.
