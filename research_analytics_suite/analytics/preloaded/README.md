<!-- ABOUT THE PACKAGE -->
## About The Package

The `preloaded` package within the `analytics` package contains preloaded transformation functions and utilities to be used within the Research Analytics Suite (RAS). This package serves as a repository for various built-in transformations that can be applied to data points in research analytics projects.

### Subpackage: transformations
The `preloaded` package includes the `transformations` subpackage, which contains specific transformation functions.
